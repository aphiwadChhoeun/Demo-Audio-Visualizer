# empty-project

Demo Audio Visualizer

## Building and running on localhost

First install dependencies:

```sh
npm install
```

To create a production build:

```sh
npm run build-prod
```

To create a development build:

```sh
npm run build-dev
```

## Running Local

```sh
npm rund dev
```

Open browser at https://localhost:3000
